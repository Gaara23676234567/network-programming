# 🏨 Hotel Reservation API (Завдання 18)

Цей проєкт є RESTful сервісом для керування бронюванням готельних номерів. Виконаний як індивідуальне завдання в межах курсу веб-розробки.

## 🛠 Технологічний стек
* **Java 21**
* **Spring Boot 3.2.0**
* **Spring Data JPA** (для роботи з БД та ORM)
* **H2 Database** (In-memory БД для зручності розробки та тестування)
* **Maven** (автоматизація збірки проєкту)

## 🚀 Функціональні можливості
Реалізовано наступні API-ендпоїнти:
1. `GET /hotels/{id}/rooms?date=YYYY-MM-DD` — Перегляд доступних номерів у конкретному готелі на обрану дату.
2. `POST /hotels/{id}/rooms/{roomId}/book` — Бронювання номера (створення запису в базі даних).
3. `DELETE /reservations/{id}` — Скасування бронювання (видалення запису за ідентифікатором).

## 📊 База даних
Використовується вбудована база даних **H2**. 
- Консоль керування доступна за адресою: `http://localhost:8080/h2-console`
- **JDBC URL:** `jdbc:h2:mem:hoteldb`
- **User:** `sa` (пароль порожній)
